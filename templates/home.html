Welcome to the Scratch clone!

This is a remake of the 2013 Scratch website, built with Django. It is not affiliated with the official Scratch website in any way.

Use the navigation bar to explore projects, view user profiles, and more!

The server will spin down after a period of inactivity. All data will be lost when the server spins down.

If you are using a modern browser,

you may need to install the Ruffle Flash Emulator.

The Website includes the emulator, but you may need to install it as a browser extension.


You can also download an older browser that supports Flash (Recommended for best compatibility), and use a version of Flash without the lockout.
